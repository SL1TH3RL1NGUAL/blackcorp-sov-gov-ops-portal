# Privacy & Security Notice — BLACK CORP SOV GOV OPS Portal

## 1. Static, Front-End Only

This portal is served as a static site (e.g., via GitHub Pages). It does not intentionally:

- Set cookies.
- Embed third-party analytics.
- Load third-party scripts or fonts.

Any logging performed by the hosting provider (e.g., IP addresses in server logs) is outside
the direct control of BlackCorp.

## 2. No Operational Data Collection via This Front Door

This front door is ceremonial and informational. It is not intended to collect operational
participant data.

If and when BlackCorp publishes a dedicated, encrypted channel for jurisdictional registration
or operations, that channel will be documented separately and governed by additional notices.

## 3. Network-Level Eavesdropping

Although this portal is intended to be served over HTTPS, no guarantees of confidentiality can
be made. Network-level monitoring by third parties (e.g., ISPs, infrastructure providers) may
still occur.

You are encouraged to:

- Access this portal from trusted networks and devices.
- Use end-to-end encrypted channels for any sensitive communication.
- Employ hardware-backed key management where possible.

## 4. Third-Party Services

This portal may be hosted on infrastructure operated by third parties (e.g., GitHub). Their
own privacy policies and practices apply to any data they collect in the course of providing
hosting and network services.

## 5. Changes to This Notice

This notice may be updated over time. Continued access to this portal after changes are
published constitutes acknowledgment of the updated notice.
